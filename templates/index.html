This site provides reports on various aspects of the data recorded in the Wait Wait... Don't Tell Me! Stats and Show Details database that don't really fit neatly into the main Stats and Show Details site.

Location

Report Description
Average Scores by Location A report listing out the average panelist score and average total score for each location the show has had a show broadcasted from; excluding the 25th Anniversary Special recorded in October 2018 due to the different scoring system used

Panelist

Report Description
Aggregate Scores Statistics and score spread based on an aggregate of all panelist scores
Appearances by Year A pivot table containing a list of panelists and the number of appearances each panelist has made each year, excluding Best Of and Repeat shows
Panel Gender Mix A report breaking down the panel gender mix for each show by year, excluding Best Of, Repeat and special shows (including pledge specials and the 25th anniversary show)
Panelist vs Panelist A report detailing how many times a panelist has ranked versus other panelists
Win Streaks A report detailing the longest outright win streak and the longest win streak (including draws) for each panelist

Scorekeeper

Report Description
Scorekeeper Introductions A report listing of all of the introductions that Bill Kurtis and other scorekeepers have used when introducing themselves at the start of each show

Show

Report Description
Original Shows A report listing each original broadcast show (thus excluding Best Of, Repeat, pledge specials and other special shows that were not broadcast in their entirety)
